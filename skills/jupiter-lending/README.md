# Jupiter Lending

Focused skill for building and managing Jupiter Lend Earn integrations.

## What does the skill cover

- **SKILL.md** - Deep integration guide for Jupiter Lend deposit/withdraw flows and risk management

| Topic | Description |
|-------|-------------|
| Architecture | Unified liquidity model, program addresses, debt ceiling |
| Endpoint Reference | All 9 Earn endpoints with methods and purposes |
| Deposit / Withdraw | Complete TypeScript flow with state refresh and confirmation |
| Mint / Redeem | Share-based alternatives for transferable position tokens |
| Position Monitoring | Positions, earnings, and token APY queries |
| Risk Management | Health factor rules, liquidation boundaries, safe withdraw pattern |
| Debt Ceiling | Partial withdrawal handling and retry strategy |
| Error Handling | HTTP errors, simulation failures, on-chain program errors |

## When to use this skill

Use when you need to build, debug, or manage Jupiter Lend Earn flows — deposits, withdrawals, position monitoring, risk management, and debt ceiling handling.
