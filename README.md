# Agent Skills

Skills for AI coding agents in the Jupiter ecosystem.

Skills follow the [Agent Skills](https://agentskills.io/) format.

## Available Skills

### migrating-lite-api

Helps agents guide you through migrating from deprecated Lite API to Jupiter API. The migration would be definitive by March 1st, 2026.

```bash
npx add-skill jup-ag/agents-skills --skill "migrating-lite-api"
```
