# Agent Skills

Skills for AI coding agents to integrate with the Jupiter ecosystem.

Skills follow the [Agent Skills](https://agentskills.io/) format.

## Available Skills

### integrating-jupiter

Helps agents integrate with the whole Jupiter Suite of APIs. 

#### Installation

```bash
npx add-skill jup-ag/agents-skills --skill "integrating-jupiter"
```

### executing-jupiter

Helps agents execute Jupiter API operations using scripts for fetch, sign, and swap execution flows.

#### Installation

```bash
npx add-skill jup-ag/agents-skills --skill "executing-jupiter"
```



## Quick Install
