# Agent Skills

Skills for AI coding agents to integrate with the Jupiter ecosystem.

Skills follow the [Agent Skills](https://agentskills.io/) format.

## Available Skills

### integrating-jupiter

Helps agents integrate with the whole Jupiter Suite of APIs.

#### Installation

```bash
npx skills add jup-ag/agent-skills --skill "integrating-jupiter"
```

### executing-ultra-swaps

Production guide for Jupiter Ultra Swap execution: order-sign-execute flow, slippage rules, error recovery, and re-quote logic.

#### Installation

```bash
npx skills add jup-ag/agent-skills --skill "executing-ultra-swaps"
```

### jupiter-lending

Production guide for Jupiter Lend Earn: deposit/withdraw flows, position monitoring, risk management, and debt ceiling handling.

#### Installation

```bash
npx skills add jup-ag/agent-skills --skill "jupiter-lending"
```
